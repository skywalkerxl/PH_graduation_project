# 介绍

本部分移动端开发独立于主端单独开发，存放于mobile文件夹中


# 开始
1.安装node

2.安装npm

3.切换至本文件夹下,

安装包依赖
> npm i

开始加载并启用热更新
> npm start

开发模式下打包
> npm run dev

上线压缩模式打包
> npm run dist


4.需要与后端测试运行时，请打包文件至dist文件夹之后，将html文件移动至templates文件夹下，其他静态文件夹移动至static文件夹下的mobile文件夹中
